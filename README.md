# UKS-DGL 空间插值实验精度调优与项目归档 (Run 11 Final)

本仓库包含了可微统一克里金神经网络系统 (Unified Kriging System with Deep Geometric Learning, UKS-DGL) 在六个经典地统计学空间仿真场景 (Scenario A - F) 下进行插值精度调优的完整代码、最佳权重与实验数据。

根据用户的最新指示，本仓库已全面扩展至 **6个仿真场景**（Scenario A ~ F）。通过对不同空间物理特性的深入剖析，并针对性地对平稳场、各向异性非平稳场、多变量场和强偏态场进行物理趋势面降阶、自适应块金以及强几何正则超参网络搜索 (HPO)，UKS-DGL 在多个场景上彻底击穿了克里金基线，并在全部神经网络架构中保持绝对领先的精度。

## 🛠️ 2. 最优超参复现参数配置表 (Best Hyperparameter Configs)

为了便于在各个场景中完美复现最佳实验结果，下表详细列出了在云端 GPU 上通过 HPO 调优锁定的最佳物理和模型参数：

| 参数项 (Hyperparameters) | 场景 A | 场景 B | 场景 E | 场景 F | 物理/工程设计说明 |
| :--- | :---: | :---: | :---: | :---: | :--- |
| **学习率 (Learning Rate)** | `0.0015` | `0.0025` | `0.0003` | `0.0002` | 控制模型梯度更新的步长尺度。 |
| **均值趋势类型 (Trend Type)** | `constant` | `quadratic` | `quadratic` | `quadratic` | A场景漂移平缓；B、E、F场景非平稳，需二次趋势面。 |
| **流体积正则 ($\lambda_{flow}$)** | `0.0010` | `0.0030` | `0.0005` | `0.0010` | 约束流网络体积收敛，对齐隐特征高斯分布。 |
| **测地几何正则 ($\lambda_{geo}$)** | `1.0e-05` | `5.0e-06` | `5.0e-06` | `5.0e-06` | Hessian 测地几何曲率平滑正则。 |
| **各向异性偏心上限 (`l2_max`)**| `0.02` | `0.20` | `0.20` | `0.20` | 控制各向异性马氏椭圆的主次轴拉伸上限。 |
| **各向同性退化 (`force_isotropic`)** | `true` | `false` | `true` | `false` | 强制局部核退化为各向同性（A、E开启）。 |
| **块金项系数 (`nugget_eps`)** | `1.0e-07` | `1.0e-06` | `5.0e-05` | `1.0e-07` | 稳定克里金核矩阵求逆。F 极偏态需极小块金以防平滑。 |
| **自监督采样数 (`num_samples`)** | `400` | `200` | `400` | `400` | 控制模型前向自监督掩码训练的观测点规模。 |
| **可逆流隐藏维度 (`flow_hidden`)** | `32` | `32` | `32` | `64` | 针对 F 偏态流形，将流网络隐藏层大幅加宽以扩容。 |
| **可逆流 Coupling 层数** | `4` | `4` | `4` | `8` | 针对 F 偏态流形，将层数翻倍以稳定强偏态流形。 |
